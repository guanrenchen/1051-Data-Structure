Maze:
	1. uses pass-by-value property to pass on the routes with possible next step (recursive)
	2. print out the route if made it to the destination, go back to 1 if didn't make it.

NoncircularQueue:
	use the front and rear index to decide the status of the queue and how to insert and delete.

CircularQueue:
	basically the same as NoncircularQueue, but uses the % operator to make the circular part possible
